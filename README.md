# StringMixer
Easy Algorithm to protect your strings

How to use it? 

--- Encrypt ---

StringMixer.Encrypt(string input);

--- Decrypt ---

StringMixer.Decrypt(string intput);

Easy no? 

Limits : 

Specials characters such as @,#/: are not supported !
